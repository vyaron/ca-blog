כוחות העל של CSS

והביקוש הגובר במתכנתי פרונטאנד מעולים

ירון ביטון
טכנולוג ראשי · 3 דקות קריאה

י ש עלייה מדהימה בשימוש בטכנולוגיות אינטרנט, ומפתחי פרונטאנד טובים הם אחד התפקידים הקשים ביותר לגיוס, CSS היא סיבה משמעותית לכך.

יש הסבורים בטעות שCSS זה בסך בכל צבעים ופיקסלים ועניינים הקשורים "רק לעיצוב" ולא לתכנות, למעשה – שימוש נכון בCSS היא מיומנות הנדסית מורכבת.

אם נעשה נכון, הכוח של CSS מאפשר לבנות מוצרים יפיפיים, שמעניקים חוויית משתמש טובה ומדויקת, להוביל את המשתמש בקלות אל הפעולה המבוקשת, ולהוסיף משחקיות ועניין.

אפליקציות מודרניות נראות נהדר ויכולות להיות מאוד אינפורמטיביות על מסך גדול, בנוסף עליהן להיות מותאמות ולהציג ממשק משתמש מינימליסטי וממוקד יותר כאשר הלקוח משתמש בטאבלט, בטלפון או על מסך שעון.

CSS טוב הוא כזה שניתן להתאים אותו בקלות גם לדרישות העסקיות המשתנות, ועדיין להישאר קטן, מודולרי ולא יקר מבחינת התחזוקה וההתפתחות לאורך חיי המוצר.

בשימוש לא נכון, ה-CSS של המוצר יהפוך לסיוט המאוחסן בקבצים, אזור מאמבו-ג'אמבו-קסם-שחור שמפתחים נמנעים ממנו, עם רגרסיות רבות והשפעה לא צפויה בעת ביצוע שינויים.
